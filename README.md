# GOH Vote Maryland

A Flutter mobile application that helps users access Maryland election resources.  
Users can quickly **check their voter registration, view the election calendar, access FAQs, find county-specific locations, and get helpful links**.


## Features

- **County Selector**: Choose your county to access localized election information.
- **Check Registration**: Navigate to a page where users can confirm their voter registration status.
- **Election Calendar**: View upcoming election dates and deadlines.
- **FAQs**: Quick access to frequently asked questions.
- **Contact Information**: Reach out to election offices directly.
- **Helpful Links**: Curated external resources for Maryland voters.
- **Locations**: Polling places, drop boxes, and other election-related locations.


## Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (>=3.0.0 recommended)
- Dart >= 3.0
- Android Studio or Xcode (for mobile builds)
